# Troubleshooting

## Common Issues

### High Temperature Warning

**Symptom**: Battery temperature exceeds 45°C.
**Solution**: Check thermal management system and coolant levels.

### SoC Drift

**Symptom**: State of Charge reading is inaccurate.
**Solution**: Recalibrate the SoC Estimator using the diagnostic tool.
