# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| < latest| :x:                |

Only the latest published version receives security updates.

## Reporting a Vulnerability

If you discover a security vulnerability in `okf-schema`, please report it
privately via email to **gaetan@xeberon.net**.

Please include:

- A description of the vulnerability
- Steps to reproduce (if applicable)
- The version(s) affected
- Any suggested remediation

You can expect an initial response within 72 hours. We will coordinate a fix
and disclosure timeline with you.

Please do **not** open public issues for security vulnerabilities.
