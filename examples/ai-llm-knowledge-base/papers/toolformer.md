# Toolformer

Toolformer enables LLMs to use search engines, calculators, calendars,
and translation systems without requiring human-annotated tool-use data.

See also [Function Calling](../concepts/function-calling.md) for the API-based
approach to tool use, and [ReAct](../concepts/react-pattern.md) for the
reasoning pattern that interleaves thought with tool invocation.
