# Attention Is All You Need

This seminal paper proposed the Transformer, which became the foundation
for modern LLMs like GPT, BERT, and T5.

The Transformer architecture underpins techniques such as
[RAG](../concepts/rag.md) and [Function Calling](../concepts/function-calling.md).
