# LangChain

LangChain provides abstractions for prompt management, output parsing,
memory, and agent orchestration across multiple LLM providers.

It is commonly used to implement patterns such as
[RAG](../concepts/rag.md) and [ReAct](../concepts/react-pattern.md).
