# Tool

Schema for developer tools, frameworks, and libraries. The `type` field must be exactly `tool` so the validator loads this schema.

- [LangChain](langchain.md) — A Python and JavaScript framework for building applications with LLMs through composable chains and agents.  [tool]
- [Ollama](ollama.md) — A lightweight tool for running open-source LLMs locally with a simple command-line interface.  [tool]
- [Pytest](pytest.md) — The standard testing framework for Python, supporting fixtures, parametrization, and plugins.  [tool]
